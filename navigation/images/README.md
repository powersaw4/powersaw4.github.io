## To Add Custom Images

1. Place image files in this folder
2. Update references in `index.html`
3. Optimize images before upload (use tinypng.com)

## Recommended Formats

- PNG for logos and icons
- JPG for photographs
- SVG for vector graphics
- WebP for best performance

## File Naming

Use lowercase with hyphens:
- `category-homes.png`
- `category-hospitality.png`
- `logo-brickit.svg`

## Image Optimization

Before deploying, optimize all images:
- Maximum width: 2000px
- Compress with tinypng.com or imageoptim.com
- Use appropriate format for content type
